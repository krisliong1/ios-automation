# iOS 自动化 AI 管理器 - Claude App 项目

## 🎯 项目说明

这是一个 iOS 自动化开发项目，集成了智能 AI 管理器来自动解决开发中的各种问题。

**AI 管理器 = 你的 AI 助手的"保姆"**
- 自动检测问题（网络、搜索、超时等）
- 自动切换 AI 提供商
- 自动修复并重试

## 📋 常见任务

### 1. 编译问题
```
Xcode 编译失败，错误代码 65
```

AI 会：
- 分析错误类型
- 搜索最新解决方案
- 提供详细步骤
- 自动处理网络问题

### 2. 虚拟机检测
```
Xcode 检测到虚拟机无法运行
```

AI 会：
- 识别虚拟机问题
- 搜索 VMHide 等方案
- 提供安装步骤

### 3. 证书问题
```
找不到开发者证书
```

AI 会：
- 诊断证书问题
- 搜索解决方案
- 提供详细步骤

### 4. 网络问题
```
搜索受限，无法访问
```

AI 会：
- 检测网络限制
- 自动切换搜索引擎
- 继续搜索

## 🐛 常见问题

### Q: 如何知道当前用的什么模式？

A: 直接说"检查状态"，会显示当前提供商

### Q: 什么时候用快速模式？

A:
- 简单问题
- 大量任务
- 需要快速响应时

### Q: 什么时候用本地模式？

A:
- 离线工作
- API 配额用完
- 需要无限制使用时

### Q: 网络慢怎么办？

A: 说"切换到本地模式"，完全离线使用
