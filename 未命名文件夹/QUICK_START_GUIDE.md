# 🚀 5 分钟快速开始 - Claude iOS App

## 📱 第一步：创建项目（1 分钟）

```
1. 打开 Claude iOS App
2. 点击 "Projects" 标签
3. 点击 "+" 创建新项目
4. 命名：iOS 自动化 AI 管理器
```

---

## 📚 第二步：添加文档（2 分钟）

点击 "Add Knowledge" 或 "📎"，添加这 3 个文档：

```
✅ docs/AI-Manager-Quick-Start.md
✅ docs/AI-Manager-Integration-Guide.md
✅ .claude-project/instructions.md
```

**在哪里找这些文件？**
- 在 GitHub 仓库中
- 或从项目文件夹中复制

---

## ⚙️ 第三步：设置说明（1 分钟）

在项目设置中，粘贴这段说明：

```markdown
iOS 自动化助手，自动解决开发问题。

使用方法：
- "检查 AI 状态"
- "帮我解决: [问题]"
- "切换到快速模式"

AI 会自动处理网络问题！
```

---

## ✅ 第四步：测试（1 分钟）

在项目对话中输入：

```
检查 AI 状态
```

看到状态信息 → 🎉 设置成功！

## 🎯 3 种模式

```
默认模式（Claude）    → 高质量回答
快速模式（DeepSeek）  → 快速响应
本地模式（Ollama）    → 离线使用
```

切换方法：
```
切换到快速模式
切换到本地模式
```
