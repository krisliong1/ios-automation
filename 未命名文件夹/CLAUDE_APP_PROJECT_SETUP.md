# 在 iOS Claude App 中设置 AI 管理器项目

## 📱 完整设置步骤

### 步骤 1: 创建项目

1. **打开 Claude iOS App**

2. **进入 Projects**
   - 点击底部的 "Projects" 标签
   - 或点击左上角菜单 → Projects

3. **创建新项目**
   - 点击右上角的 "+" 按钮
   - 或点击 "Create Project"

4. **填写项目信息**
   ```
   项目名称: iOS 自动化 AI 管理器
   描述: 智能 iOS 开发助手，自动解决编译、网络等问题
   ```

---

### 步骤 2: 添加知识库文档

点击 "Add Knowledge" 或 "📎"，添加以下文档：

#### 必需文档（3 个）

1. **AI-Manager-Quick-Start.md**
   - 路径: `docs/AI-Manager-Quick-Start.md`
   - 内容: 快速开始指南

2. **AI-Manager-Integration-Guide.md**
   - 路径: `docs/AI-Manager-Integration-Guide.md`
   - 内容: 完整使用指南

3. **.claude-project/instructions.md**
   - 路径: `.claude-project/instructions.md`
   - 内容: 项目使用说明（本指南）

#### 可选文档

4. **AI-Manager-Claude-App-Usage.md**
   - 路径: `docs/AI-Manager-Claude-App-Usage.md`
   - 内容: App 专用使用方法

---

### 步骤 3: 设置项目说明

在项目设置中，添加自定义说明：

```markdown
# 项目说明

这是一个 iOS 自动化开发助手，集成 AI 管理器。

## 快速使用

在对话中直接说：

- "检查 AI 状态" - 查看系统状态
- "帮我解决: [问题]" - 自动解决问题
- "切换到快速模式" - 切换提供商

## 提供商

- 默认模式（Claude）- 高质量
- 快速模式（DeepSeek）- 快速响应
- 本地模式（Ollama）- 离线使用

## 自动功能

AI 管理器会自动：
- 检测网络问题并切换提供商
- 检测搜索限制并切换搜索引擎
- 监控系统健康状态

开始使用：直接描述你的问题即可！
```

## 🎯 推荐设置（可选但推荐）

### 1. 添加到主屏幕

如果经常使用这个项目：

1. 在项目列表中
2. 长按项目卡片
3. 选择 "Add to Home Screen"

现在可以直接从主屏幕打开！

### 2. 设置 Siri 快捷方式

创建快捷指令：

```
快捷指令名称: AI 管理器
操作:
  1. 打开 App: Claude
  2. 打开项目: iOS 自动化 AI 管理器
```

使用：
```
"嘿 Siri，打开 AI 管理器"
```

### 3. 固定到顶部

在项目列表中：
1. 向左滑动项目
2. 点击 "📌 Pin"

项目会固定在列表顶部。

## 💡 实际使用示例

### 示例 1: 解决编译问题

**你的输入**:
```
Xcode 编译失败，提示：
"Could not find developer disk image"
```

**AI 的响应**:
```
🔍 正在分析问题...
检测到：开发者磁盘镜像问题

🌐 搜索最新解决方案...
找到 3 个解决方案

【最佳方案】下载匹配的 iOS 支持文件

步骤：
1. 查看设备 iOS 版本
   设置 → 通用 → 关于本机

2. 下载对应版本的支持文件
   https://github.com/filsv/iPhoneOSDeviceSupport

3. 解压到 Xcode 目录
   ~/Library/Developer/Xcode/iOS DeviceSupport/

4. 重启 Xcode

需要我详细解释哪一步吗？
```

```

   ✅ 搜索成功
```

## 🔧 故障排除

### 问题 1: AI 没有显示状态

**可能原因**: 项目知识库未正确加载

**解决方法**:
1. 检查文档是否已添加到项目
2. 重新打开项目
3. 或重新创建项目

### 问题 2: AI 回复不符合预期

**可能原因**: 项目说明不够清晰

**解决方法**:
1. 更新项目说明
2. 明确告诉 AI 你的需求
3. 使用更具体的描述

### 问题 3: 找不到文档

**解决方法**:
确保以下文件存在：
```
docs/AI-Manager-Quick-Start.md
docs/AI-Manager-Integration-Guide.md
.claude-project/instructions.md
```
